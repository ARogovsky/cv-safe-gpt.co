# Brandfetch Icon Fetcher

Автоматический инструмент для получения SVG иконок брендов через [Brandfetch API](https://brandfetch.com/).

## Требования

1. **Node.js** 18+ установлен
2. **Brandfetch API Key** - получите бесплатно на [brandfetch.com/developers](https://brandfetch.com/developers)

## Быстрый старт

### 1. Получите API ключ

1. Зарегистрируйтесь на https://brandfetch.com/developers
2. Создайте аккаунт (бесплатно)
3. Скопируйте ваш API ключ из дашборда

### 2. Установите API ключ

```bash
export BRANDFETCH_API_KEY=your_api_key_here
```

Или добавьте в `.env` файл:

```bash
echo "BRANDFETCH_API_KEY=your_api_key_here" >> .env.local
```

### 3. Запустите скрипт

```bash
node scripts/fetch-brand-icons.mjs openai anthropic vercel
```

- 🎯 Получение SVG логотипов по домену или названию бренда
- 🎨 Автоматическое извлечение основного цвета бренда
- 📦 Генерация готового TypeScript кода для `tech-icons.ts`
- 💾 Сохранение полных SVG файлов для справки
- 📊 JSON экспорт всех данных
- 🔄 Пакетная обработка из файла
- 💬 Интерактивный режим

## Использование

### 1. Прямое указание брендов

```bash
./scripts/fetch-brand-icons openai anthropic vercel
```

### 2. Из файла

```bash
./scripts/fetch-brand-icons --file scripts/brands-to-fetch.txt
```

### 3. Интерактивный режим

```bash
./scripts/fetch-brand-icons --interactive
```

Затем вводите бренды по одному на строку, завершите вводом `done` или Ctrl+D.

## Формат входных данных

Скрипт принимает:
- **Домены**: `openai.com`, `github.com`
- **Названия брендов**: `openai`, `github`
- **Тикеры**: `AAPL`, `TSLA`
- **ISIN коды**: `US0378331005`

## Выходные файлы

Все файлы сохраняются в `scripts/brand-icons-output/`:

### 1. `icons.ts` - TypeScript код

```typescript
// Generated by scripts/fetch-brand-icons
// Date: 2026-05-02T...

// OpenAI (openai.com)
icons['openai'] = {
  color: '#412991',
  path: 'M22.282 9.821a5.985...',
}

// Anthropic (anthropic.com)
icons['anthropic'] = {
  color: '#D97757',
  path: 'M17.3041 3.541h-3.6718...',
}
```

### 2. `icons.json` - Полные данные

```json
{
  "generated": "2026-05-02T...",
  "count": 2,
  "icons": [
    {
      "name": "openai",
      "displayName": "OpenAI",
      "domain": "openai.com",
      "color": "#412991",
      "path": "M22.282 9.821...",
      "type": "icon"
    }
  ]
}
```

### 3. `{brand}.svg` - Полные SVG файлы

Оригинальные SVG файлы для каждого бренда.

## Интеграция с проектом

1. Запустите скрипт:
   ```bash
   ./scripts/fetch-brand-icons --file scripts/brands-to-fetch.txt
   ```

2. Откройте сгенерированный файл:
   ```bash
   cat scripts/brand-icons-output/icons.ts
   ```

3. Скопируйте нужные иконки в `src/tech-icons.ts`

4. Добавьте алиасы при необходимости:
   ```typescript
   icons['openai api'] = icons['openai']
   icons['openai embeddings'] = icons['openai']
   ```

## API Лимиты

Brandfetch API имеет следующие лимиты для бесплатного использования:
- **1000 запросов/месяц** с бесплатным API ключом
- **Rate limit**: ~2 запроса/секунду

Скрипт автоматически добавляет задержку 500ms между запросами.

Для больших объемов рассмотрите платный план: https://brandfetch.com/pricing

## Troubleshooting

### API ключ не установлен

```
⚠️  Warning: BRANDFETCH_API_KEY not set
```

**Решение**: Установите переменную окружения:
```bash
export BRANDFETCH_API_KEY=your_key_here
```

### Бренд не найден

```
✗ Error: Brand not found: mybrand
```

**Решение**: Попробуйте использовать полный домен (`mybrand.com`) или проверьте написание.

### SVG path не извлечен

```
✗ Could not extract SVG path
```

**Решение**: Некоторые SVG используют сложные структуры. Проверьте сохраненный `.svg` файл и извлеките path вручную.

### HTTP 429 (Too Many Requests)

**Решение**: Подождите несколько минут или увеличьте задержку в коде (строка с `setTimeout`).

## Структура данных Brandfetch

API возвращает:

```json
{
  "name": "OpenAI",
  "domain": "openai.com",
  "logos": [
    {
      "type": "icon",
      "theme": "light",
      "formats": [
        {
          "src": "https://...",
          "format": "svg",
          "background": "transparent"
        }
      ]
    }
  ],
  "colors": [
    {
      "hex": "#412991",
      "type": "accent"
    }
  ]
}
```

## Приоритет типов логотипов

Скрипт выбирает логотипы в следующем порядке:
1. **icon** - квадратная иконка (лучше для tech-icons)
2. **symbol** - абстрактный символ бренда
3. **logo** - горизонтальный логотип
